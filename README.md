## Purpose

A controlled three-paradigm comparison of algorithmic approaches to next-day SPY directional prediction, evaluated under a common temporal holdout framework using institutional WRDS CRSP data. Three fundamentally different computational paradigms — stochastic modelling (Geometric Brownian Motion), evolutionary computation (Genetic Algorithm via DEAP), and supervised machine learning (XGBoost) — are evaluated on the same held-out test set under identical metrics to determine whether any can exceed a naive majority-class baseline.

## Configuration

## What is tracked vs ignored

**Version-controlled:** source code (`src/`), configuration (`config/`), documentation (`docs/`), `CLAUDE.md`, `README.md`, `requirements.txt`, `.gitignore`.

**Not version-controlled (`.gitignore`):** `data/` (raw, interim, processed CSVs), `results/` (predictions, tables, figures), `.venv/`, `__pycache__/`, `.env`. These are generated artefacts that must be regenerated by running the pipeline.

## Headline results

| Model | Test Accuracy | vs Baseline | MCC | Prediction Pattern |
|---|---|---|---|---|
| Majority class | 57.5% | — | — | Constant "up" (100%) |
| GeoBM | 57.5% | +0.0pp | 0.000 | Constant "up" (100%) |
| GA | 56.7% | -0.8pp | -0.014 | Near-constant "up" (96.4%) |
| XGBoost | 53.1% | -4.4pp | -0.037 | Up-biased (80.4%) |

No model exceeded the naive majority-class baseline. McNemar's test confirms the GeoBM vs XGBoost difference is statistically significant (p=0.034). All MCC values are near zero, indicating no discriminative ability.
